## Cadastro-Login

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de desenvolvimento Full Stack, utilizando React no frontend e Node.js com Express no backend. 
A aplicação permite o cadastro e autenticação de usuários, realizando validações de dados e armazenamento das informações em banco de dados SQLite.

O projeto foi criado como parte dos estudos em Engenharia de Software, abordando conceitos de APIs REST
integração entre frontend e backend, organização em arquitetura MVC e boas práticas de desenvolvimento web.
